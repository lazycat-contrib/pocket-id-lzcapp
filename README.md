# pocket-id-lzcapp

> note: after install goto "https://domain.com/setup" to setup your pocket-id-lzcapp

项目已升级为 LPK v2（独立 `package.yml` 与 `lzc-manifest.yml`，最低系统版本 1.5.0）。每天 23:00 UTC 检查 `ghcr.io/pocket-id/pocket-id` 稳定版本，自动构建版本化 Release，并提交懒猫官方商店和喵喵私有商店。

GitHub Secrets：官方商店使用 `LAZYCAT_TOKEN`（或兼容凭据）；喵喵商店使用 `APPSTORE_URL`、`APPSTORE_TOKEN`，可选 `APP_ID` 和 `PRIVATE_STORE_GROUP_CODES`。
